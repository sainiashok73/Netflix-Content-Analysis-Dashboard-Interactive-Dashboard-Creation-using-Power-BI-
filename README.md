# Netflix-Content-Analysis-Dashboard-Interactive-Dashboard-Creation-using-Power-BI-

## Description
I developed a Netflix Content Analysis Dashboard using Power BI to analyze Netflix movies and TV shows data and generate meaningful insights. The dashboard visualizes content distribution, genres, country-wise content availability, and content trends over time using interactive charts, filters, KPI cards, and data visualization techniques.

## Dataset
- <a href="https://github.com/sainiashok73/Netflix-Content-Analysis-Dashboard-Interactive-Dashboard-Creation-using-Power-BI-/blob/main/netflix_titles.csv">Datset</a>

## Project Objective
The objective of this project is to analyze Netflix content data to understand content distribution, genre popularity, country-wise content production, and content growth trends. The dashboard helps visualize streaming platform insights and supports better understanding of entertainment content patterns.

## Process
Collected hotel booking dataset from Kaggle.
Imported and cleaned the raw dataset in MS Excel.
Removed duplicate and missing values for accurate analysis.
Performed data transformation and formatting.
Created Pivot Tables to summarize booking and cancellation data.
Designed KPI cards for Total Bookings and Total Cancellations.
Created charts for hotel types, guest types, room status, and monthly cancellation trends.
Added slicers for year-wise interactive filtering.
Built an interactive dashboard for easy data visualization and analysis.
Generated insights to understand cancellation patterns and improve business decisions.
